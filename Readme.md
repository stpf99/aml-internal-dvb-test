# Amlogic DVB Support for LibreELEC (S905D / GXL)

## Status: EXPERIMENTAL - AI-PORTED FROM COREELEC 22

### Supported Hardware
- Mecool M8S Plus DVB-S2X (S905D)
- M88RS6060 frontend
- Serial & Parallel TS modes

### Features
- Full DVB-S2X support
- Hardware demux
- Simultaneous USB + built-in tuner (with kernel 6.x)
- Mainline kernel 6.12 compatible
